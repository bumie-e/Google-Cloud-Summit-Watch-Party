About the Gemini CLI workshop

This short lab introduces the Gemini CLI: installing, authenticating, running queries, saving outputs, and simple scripting patterns to integrate models into reproducible workflows. It will focus on practical, reproducible commands and small examples students can run on their laptops.

What you'll learn

  • Install and configure Gemini CLI
  • Run text generation & retrieval commands
  • Use CLI output in scripts (bash / python snippets provided)
  • Best practices for reproducible prompts and token/usage considerations

Who should attend

Students with basic command-line familiarity. No prior cloud account required for the curated, offline-friendly demos.
